# projeto-oficina facil
Este meu primeiro projeto iniciando na programação que fala muito com a minha experiencia de vida e profissional. A ideia é facilitar o dia a dia de oficinas e mecanicos aonde mutos não tem tempo e nem ideia de como mexer em sites mais complicados meu projeto foca em ser direto e facil 
# Sistema de Oficina

Sistema simples para oficinas registrarem veículos por placa e gerar orçamento.

### Funcionalidades

- Cadastro de cliente
- Cadastro de peças
- Cálculo automático do serviço
- Busca por placa
- Salvamento no navegador

### Tecnologias

- HTML
- CSS
- JavaScript
